# Expense Tracker Auth App

A responsive Expense Tracker web application built with HTML, CSS, and JavaScript featuring user authentication, expense management, filtering, and real-time total calculation using localStorage.

---
## 🚀 Features

### 🔐 Authentication
- User Signup
- User Login
- Session-based access using localStorage
- Logout functionality
- Route protection (Dashboard access only after login)

### 💰 Expense Management
- Add new expenses
- Select category
- Add amount & date
- Delete individual expenses
- Real-time total calculation

### 🔍 Filtering
- Filter expenses by category


### 📱 Responsive Design
- Mobile-friendly layout
- Clean modern UI
- Smooth hover effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Responsive Design + Flexbox + Grid)
- JavaScript (Vanilla JS)
- localStorage API
